Privacy Aware Location Service

GPS is the most common approach to locate a smartphone-users. The GPS location is determined in Android by Google services. Google stores user profiles about their movement. Thus privacy is not protected!

This application uses location services to treat the data anonymous. These can be installed on your smartphone and be integrated into your own applications.

The Android application is developed and pull requests are welcome!

Used Location-Services

  • Mozilla Location Service
  • OpenWLANMap
  • OpenCellID
  • OpenBMap

Get the application

  • Download the latest release: pals_v1.0.apk

  • Install App on your Smartphone.

and use the code below in your own app

Use in external application

final Intent serviceIntent = new Intent("com.bestog.pals.Service");
final ServiceConnection serviceConnection = new ServiceConnection() { ... };
BroadcastReceiver broadcastReceiver = new BroadcastReceiver() {
  @Override
  public void onReceive(Context context, Intent intent) {
    Bundle bundle = intent.getExtras();
    float lat = bundle.getFloat("lat");
    float lon = bundle.getFloat("lon");
    // ...
    // your code
    // ...
    activity.unregisterReceiver(this);
    activity.stopService(serviceIntent);
    activity.unbindService(serviceConnection);
  }
};
activity.bindService(serviceIntent, serviceConnection, Context.BIND_AUTO_CREATE);
activity.registerReceiver(broadcastReceiver, new IntentFilter("com.bestog.pals.receiver"));
